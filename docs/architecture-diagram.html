Saga Pattern — Order Processing Workflow

Step 1: Create Order
Service: Shop
Validate stock, insert order, deduct stock
Fail: Rollback, no changes
→
Step 2: Confirm
Service: Supplier
Status: pending → confirmed
Reject: Cancel + restore stock
→
Step 3: Payment
Service: Supplier
Insert payment, status → paid
Fail: Cancel + restore stock
→
Complete ✓
Order: paid
Payment recorded
Stock deducted
